HTML - HyperText Markup Language

HTML is the standard markup language for documents designed to be displayed in a web browser. It is often assisted by technologies such as CSS and JavaScript.[1]

The HyperText Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser.

Example HTML:

<!DOCTYPE html>
<html>
  <head><title>My HTML Page</title></head>
  <body><h1>Welcome</h1></body>
</html>

Keyboard shortcut to view HTML source: Ctrl + U

HTML Terminology

Tag
An element like <p> or <a>
Attribute
Provides additional information about an element
Element
The combination of opening tag, content, and closing tag

CSS - Cascading Style Sheets

CSS is a style sheet language used for describing the look and formatting of a document written in HTML.[2]

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}
Basic CSS Properties
PropertyDescription
colorText color
marginOuter spacing
paddingInner spacing

JavaScript - The Language of the Web

JavaScript is a high-level programming language that enables interactive web pages. It is a core technology of the World Wide Web.[3]
