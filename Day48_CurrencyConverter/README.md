# 💱 Currency Converter App  

A simple yet smart **Currency Converter** built with **HTML, CSS, and JavaScript**.  
This project is part of my **100Days100Projects** challenge.  

---

## 🚀 Features  
- Convert between **all world currencies** 🌍  
- Live exchange rates via **ExchangeRate API**  
- Clean **dark theme UI**  
- Fun **money falling animation** (visual effect)  
- Fully **responsive design** (desktop + mobile)  

---

## 🛠 Concepts Used  
- **DOM Manipulation** (dynamic dropdown + user input)  
- **Fetch API** (live data from ExchangeRate API)  
- **Event Handling** (convert button, inputs, select change)  
- **CSS Animations** (money falling background effect)  
- **Dark Theme Styling** (modern design approach)
