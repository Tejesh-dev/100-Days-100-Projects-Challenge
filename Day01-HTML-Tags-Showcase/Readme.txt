# 🏁 Day 1: HTML Tags Showcase

This project is a complete learning guide for beginners that demonstrates the use of major HTML tags in one clean page. Each section explains why the tag is used and shows a real example.

## 🔍 Tags Included
- Semantic Tags: `<header>`, `<main>`, `<footer>`, `<nav>`, `<section>`, `<article>`, `<aside>`
- Media Tags: `<img>`, `<figure>`, `<figcaption>`
- Forms: `<form>`, `<input>`, `<textarea>`, `<label>`
- Lists: `<ul>`, `<ol>`, `<li>`
- Table: `<table>`, `<thead>`, `<tbody>`, `<th>`, `<td>`
- Inline tags: `<b>`, `<i>`, `<u>`, `<mark>`, `<code>`, `<small>`
- Meta, Title, and more!

## 📁 Project Structure
- `index.html`: Main HTML page with all tags used and explained
- No CSS used — this is 100% HTML learning project

## 🚀 What I Learned
- Semantic structure of HTML
- Purpose and usage of each core tag
- How to build clean HTML-only pages for practice
